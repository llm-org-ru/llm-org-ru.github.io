# Политика конфиденциальности MeetLLM

Дата последнего обновления: **2026-01-16**

## 1. Общие положения

MeetLLM — это Telegram-бот, предоставляющий доступ к функциям искусственного интеллекта.  
Мы уважаем конфиденциальность пользователей и стремимся обрабатывать минимально необходимый объём данных для корректной работы сервиса.

Используя бота MeetLLM, вы соглашаетесь с настоящей Политикой конфиденциальности.

---

## 2. Какие данные мы собираем

Бот может обрабатывать следующие данные:

### 2.1. Данные Telegram
- Telegram ID пользователя  
- Имя пользователя (username), если доступно  
- ID чата  

Эти данные используются исключительно для:
- идентификации пользователя,
- работы подписок и лимитов,
- корректной работы команд бота.

### 2.2. Контент сообщений
- Текстовые сообщения, отправленные боту  
- Изображения и файлы, отправленные пользователем **по его инициативе**

Сообщения используются **только** для формирования ответа ИИ и **не анализируются вручную**.

---

## 3. Хранение данных

- История диалогов может временно храниться для поддержания контекста общения.
- Данные о подписке (тип подписки, срок действия, лимиты) хранятся локально в базе данных бота.
- Платёжные данные **не хранятся** ботом напрямую.

---

## 4. Платежи и сторонние сервисы

Для обработки платежей используется сторонний сервис **Crypto Pay** (Telegram).

MeetLLM **не получает и не хранит**:
- номера кошельков,
- платёжные реквизиты,
- данные транзакций сверх необходимого статуса оплаты (оплачено / не оплачено).

Платежи обрабатываются согласно политике конфиденциальности Crypto Pay.

---

## 5. Передача данных третьим лицам

Для генерации ответов используются сторонние API-провайдеры ИИ.

Передаваемые данные:
- текст запроса,
- изображения (если пользователь их отправил).

Данные передаются **исключительно для генерации ответа** и не используются для рекламы или профилирования пользователей.

MeetLLM не продаёт и не передаёт персональные данные третьим лицам.

---

## 6. Безопасность данных

Мы принимаем разумные технические меры для защиты данных:
- ограниченный доступ к базе данных,
- отсутствие публичного доступа к конфигурационным файлам,
- минимизация хранимых данных.

Однако ни один способ передачи данных через интернет не может быть на 100% безопасным.

---

## 7. Срок хранения данных

- Контекст диалогов хранится ограниченное время и может быть удалён автоматически.
- Данные подписки хранятся до окончания срока действия и/или в рамках технической необходимости.

---

## 8. Права пользователя

Пользователь имеет право:
- прекратить использование бота в любой момент,
- очистить историю диалога с помощью встроенных команд бота,
- отказаться от платной подписки после окончания её срока.

---

## 9. Изменения политики

Администрация MeetLLM может обновлять настоящую Политику конфиденциальности.  
Актуальная версия всегда применяется с момента публикации.
